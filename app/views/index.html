API Basejump: Timestamp microservice

by Garrett

User Stories

  1. I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does, it returns both the Unix timestamp and the natural language form of that date.

Example Usage


    https://efficacious-bush.glitch.me/api/timestamp/December%2028,%202012
    https://efficacious-bush.glitch.me/api/timestamp/1356652800000

Example Output


    { "unix": 1356652800000, "natural": "Fri, 28 Dec 2012 00:00:00 GMT" }
